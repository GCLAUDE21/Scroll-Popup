# SCROLL POPUP

Projet interactif réalisé dans le cadre de ma formation Développeur Web.

**Demo live**
[Voir la page](https://gclaude21.github.io/Scroll-Popup/)

## Objectif

Créer des animations déclenchées au scroll en JavaScript natif : une navbar qui rétrécit, une image qui apparaît en translation, et une popup qui s'affiche une seule fois à un seuil de scroll défini, avec fermeture au clic.

## Compétences travaillées

- Manipulation du DOM avec JavaScript natif
- Gestion des événements (scroll, click)
- Calcul de progression de scroll
- Animations CSS pilotées par JavaScript (opacity, transform)
- Logique booléenne (affichage unique avec playOnce)

## Stack technique

JavaScript natif. HTML natif. CSS natif. Aucun framework ou bibliothèque externe.
